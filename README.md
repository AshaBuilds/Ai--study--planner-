# AI Study Planner (PartyRock App)

## 📖 Overview
This is an AI-powered Study Planner built using AWS PartyRock.  
It helps users generate personalized study schedules, structured timetables, and study strategies based on their input.

## ⚙️ Features
- Generates study timetables automatically
- Creates personalized study plans based on subject and available time
- Provides study tips and techniques
- Includes revision and break planning

## 🧠 How it works
Users enter:
- Subject(s)
- Available study time
- Learning goals

The AI then generates a structured study plan using prompt-based AI logic in PartyRock.
